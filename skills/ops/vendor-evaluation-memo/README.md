# Vendor Evaluation Memo

Evaluate a vendor option using criteria, risks, and adoption fit.

## Quick View

- Category: `ops`
- Maturity: `experimental`
- Tier: `seeded`
- Evaluation status: `unbenchmarked`

## What This Skill Does

Vendor selection often leans on opinion unless the team captures criteria and tradeoffs explicitly.

## When To Reach For It

- A vendor choice needs a structured evaluation memo.
- Stakeholders need an explicit record of vendor tradeoffs.

## What You Should Provide

- vendor options
- requirements
- constraints
- risks

## What You Should Get Back

- evaluation memo
- criteria comparison
- recommendation
