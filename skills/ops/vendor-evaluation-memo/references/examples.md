# Examples

## Existing workflow under pressure

### Scenario

A team needs vendor evaluation memo support for a real deadline instead of a brainstorming session.

### Prompt

Use Vendor Evaluation Memo to produce a vendor evaluation memo with criteria, tradeoffs, recommendation, and follow-up diligence questions for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a vendor evaluation memo with criteria, tradeoffs, recommendation, and follow-up diligence questions with explicit structure.
- Call out missing information and the next decision.
