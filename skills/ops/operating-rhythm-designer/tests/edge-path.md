# Edge Path Tests

## Late metrics edge path

### Prompt

Run Operating Rhythm Designer when metrics arrive too late to support the weekly review cadence leaders want.

### Checks

- Match cadence to signal freshness.
- Avoid rituals that recycle stale information.
- Preserve decision usefulness rather than calendar neatness.

## Forum sprawl edge path

### Prompt

Run Operating Rhythm Designer when the org already has too many recurring reviews and nobody wants to delete any of them.

### Checks

- Treat forum removal as part of the design.
- Collapse redundant rituals directly.
- Keep the operating model lighter, not just renamed.
