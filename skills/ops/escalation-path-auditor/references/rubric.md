# Rubric

| Criterion | Strong | Weak |
| --- | --- | --- |
| Signal clarity | Separates real issues from noise and ranks them for leaders. | Blends minor updates with major risks. |
| Cadence readiness | Fits recurring operating rhythms and recurring stakeholder expectations. | Feels like a one-off note with no repeatable structure. |
| Executive usefulness | Translates data into decisions, owners, and follow-ups. | Summarizes activity without showing implications. |

## Certification Note

Certified status requires benchmark coverage, score threshold, and `manual_review: passed`. It is not granted by structure alone.
