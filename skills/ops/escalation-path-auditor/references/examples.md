# Examples

## Existing workflow under pressure

### Scenario

A team needs escalation path auditor support for a real deadline instead of a brainstorming session.

### Prompt

Use Escalation Path Auditor to produce an escalation audit with trigger gaps, owner gaps, timing issues, and improvement recommendations for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce an escalation audit with trigger gaps, owner gaps, timing issues, and improvement recommendations with explicit structure.
- Call out missing information and the next decision.
