# Examples

## Existing workflow under pressure

### Scenario

A team needs hiring loop summarizer support for a real deadline instead of a brainstorming session.

### Prompt

Use Hiring Loop Summarizer to produce a hiring loop summary with rubric-aligned findings, open risks, and final decision framing for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a hiring loop summary with rubric-aligned findings, open risks, and final decision framing with explicit structure.
- Call out missing information and the next decision.
