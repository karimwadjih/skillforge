# Examples

## Existing workflow under pressure

### Scenario

A team needs codebase dependency mapper support for a real deadline instead of a brainstorming session.

### Prompt

Use Codebase Dependency Mapper to produce a dependency map with critical paths, integration surfaces, and maintenance hotspots for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a dependency map with critical paths, integration surfaces, and maintenance hotspots with explicit structure.
- Call out missing information and the next decision.
