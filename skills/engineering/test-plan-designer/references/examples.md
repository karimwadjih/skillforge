# Examples

## Existing workflow under pressure

### Scenario

A team needs test plan designer support for a real deadline instead of a brainstorming session.

### Prompt

Use Test Plan Designer to produce a test plan with coverage goals, case types, and verification priorities for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a test plan with coverage goals, case types, and verification priorities with explicit structure.
- Call out missing information and the next decision.
