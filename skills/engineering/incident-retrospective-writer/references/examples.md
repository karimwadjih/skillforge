# Examples

## Existing workflow under pressure

### Scenario

A team needs incident retrospective writer support for a real deadline instead of a brainstorming session.

### Prompt

Use Incident Retrospective Writer to produce an incident retrospective with impact, timeline, contributing factors, lessons, and follow-up actions for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce an incident retrospective with impact, timeline, contributing factors, lessons, and follow-up actions with explicit structure.
- Call out missing information and the next decision.
