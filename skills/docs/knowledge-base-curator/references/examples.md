# Examples

## Existing workflow under pressure

### Scenario

A team needs knowledge base curator support for a real deadline instead of a brainstorming session.

### Prompt

Use Knowledge Base Curator to produce a knowledge base curation plan with grouping, pruning, ownership notes, and update priorities for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a knowledge base curation plan with grouping, pruning, ownership notes, and update priorities with explicit structure.
- Call out missing information and the next decision.
