# Knowledge Base Curator

Curate a knowledge base by grouping, pruning, and clarifying article sets.

## Quick View

- Category: `docs`
- Maturity: `experimental`
- Tier: `seeded`
- Evaluation status: `unbenchmarked`

## What This Skill Does

Knowledge bases become unusable gradually, often through duplication and unclear ownership.

## When To Reach For It

- A knowledge base needs structure and cleanup.
- Support or enablement content is difficult to navigate.

## What You Should Provide

- article set
- audience
- content problems

## What You Should Get Back

- curation plan
- article grouping
- cleanup recommendations

## Repository Notes

- Main instructions live in `SKILL.md`.
- Machine-readable metadata lives in `skill.json`.
- Examples, rubric, edge cases, and tests live under `references/` and `tests/`.

## Quality Position

This Skill is part of the seeded breadth layer. It is intentionally lighter than the developed and flagship tiers.
