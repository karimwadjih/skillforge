# Failure Case Tests

## Narrative-over-operator failure probe

### Prompt

What does a poor Runbook Author output look like when it explains the system well but fails as an operator document?

### Checks

- Identify the failure mode as explanatory prose replacing executable guidance.
- Explain why that becomes dangerous during operations.
- State how the skill should restore ordered steps, verification, and containment.

## Hidden prerequisite failure probe

### Prompt

What does a poor Runbook Author output look like when a critical permission or precheck is buried halfway through the procedure?

### Checks

- Identify the failure mode as prerequisite burial.
- Explain how it creates operator risk.
- State how the skill should front-load prerequisites and privileges.
