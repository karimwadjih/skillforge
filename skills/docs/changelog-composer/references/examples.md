# Examples

## Existing workflow under pressure

### Scenario

A team needs changelog composer support for a real deadline instead of a brainstorming session.

### Prompt

Use Changelog Composer to produce a changelog draft with grouped changes, highlights, impact notes, and upgrade cautions for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a changelog draft with grouped changes, highlights, impact notes, and upgrade cautions with explicit structure.
- Call out missing information and the next decision.
