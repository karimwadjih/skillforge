# Examples

## Existing workflow under pressure

### Scenario

A team needs release note editor support for a real deadline instead of a brainstorming session.

### Prompt

Use Release Note Editor to produce a release-note edit with clearer structure, audience framing, and upgrade guidance for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a release-note edit with clearer structure, audience framing, and upgrade guidance with explicit structure.
- Call out missing information and the next decision.
