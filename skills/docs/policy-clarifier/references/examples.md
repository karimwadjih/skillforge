# Examples

## Existing workflow under pressure

### Scenario

A team needs policy clarifier support for a real deadline instead of a brainstorming session.

### Prompt

Use Policy Clarifier to produce a clarified policy document with definitions, practical guidance, and open question flags for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a clarified policy document with definitions, practical guidance, and open question flags with explicit structure.
- Call out missing information and the next decision.
