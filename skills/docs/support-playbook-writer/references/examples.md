# Examples

## Existing workflow under pressure

### Scenario

A team needs support playbook writer support for a real deadline instead of a brainstorming session.

### Prompt

Use Support Playbook Writer to produce a support playbook with issue classification, response guidance, escalation steps, and resolution notes for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a support playbook with issue classification, response guidance, escalation steps, and resolution notes with explicit structure.
- Call out missing information and the next decision.
