# Examples

## Existing workflow under pressure

### Scenario

A team needs environment drift checker support for a real deadline instead of a brainstorming session.

### Prompt

Use Environment Drift Checker to produce an environment drift report with changed items, significance, and verification priorities for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce an environment drift report with changed items, significance, and verification priorities with explicit structure.
- Call out missing information and the next decision.
