# Examples

## Existing workflow under pressure

### Scenario

A team needs capacity planning brief support for a real deadline instead of a brainstorming session.

### Prompt

Use Capacity Planning Brief to produce a capacity planning brief with assumptions, stress points, thresholds, and recommended actions for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a capacity planning brief with assumptions, stress points, thresholds, and recommended actions with explicit structure.
- Call out missing information and the next decision.
