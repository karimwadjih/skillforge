# Examples

## Existing workflow under pressure

### Scenario

A team needs infra change explainer support for a real deadline instead of a brainstorming session.

### Prompt

Use Infra Change Explainer to produce an infrastructure change explainer with rationale, affected components, impact, and follow-up checks for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce an infrastructure change explainer with rationale, affected components, impact, and follow-up checks with explicit structure.
- Call out missing information and the next decision.
