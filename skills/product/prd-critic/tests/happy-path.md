# Happy Path Tests

## Bloated scope review happy path 1

### Prompt

Review this PRD for scope bloat, hidden assumptions, and readiness for implementation.

### Checks

- Identify where the document mixes separate bets.
- Show which requirements are vague or untestable.
- Recommend the highest-leverage revisions.

## Leadership alignment happy path 2

### Prompt

Critique the PRD with a focus on decision quality, tradeoffs, and risk communication.

### Checks

- Separate product logic from presentation polish.
- Highlight decisions leadership still needs to make.
- Explain what evidence is missing.
