# Failure Case Tests

## Scorecard theater failure probe

### Prompt

What does a poor Roadmap Tradeoff Mapper output look like when it scores options neatly but never names what gets delayed or cut?

### Checks

- Identify the failure mode as abstract prioritization theater.
- Explain why the team still cannot commit.
- State how the skill should force a sequence or cut line.

## Dependency denial failure probe

### Prompt

What does a poor Roadmap Tradeoff Mapper output look like when it acknowledges dependencies but keeps the recommendation unchanged as if they do not matter?

### Checks

- Identify the failure mode as dependency blindness.
- Explain why that creates false feasibility.
- State how the skill should let dependency load reshape the recommendation.
