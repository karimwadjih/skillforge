# Edge Path Tests

## Shared bottleneck edge path

### Prompt

Run Roadmap Tradeoff Mapper when every candidate bet relies on the same data-platform team for the next two months.

### Checks

- Model the bottleneck explicitly.
- Show how it changes sequencing rather than only desirability.
- Avoid pretending the options can all proceed independently.

## Uneven estimate fidelity edge path

### Prompt

Run Roadmap Tradeoff Mapper when one option is precisely estimated and another is still rough but strategically important.

### Checks

- State the asymmetry in estimate quality.
- Avoid false precision in the comparison.
- Keep the recommendation honest about confidence.
