# Rubric

| Criterion | Strong | Weak |
| --- | --- | --- |
| Option comparability | Compares options on the same dimensions so the tradeoff is real rather than rhetorical. | Changes evaluation criteria until every option sounds defensible. |
| Dependency honesty | Shows where shared teams, platform work, or timing constraints actually shape the decision. | Mentions dependencies without letting them change the recommendation. |
| Decision force | Leaves the team with a recommendation, sequence, or explicit cut line. | Produces a balanced memo that still avoids choosing. |

## Certification Note

Certified status requires benchmark coverage, score threshold, and `manual_review: passed`. It is not granted by structure alone.
