# Edge Path Tests

## Fluid design edge path

### Prompt

Run Feature Spec Refiner when key design details are still moving but engineering planning cannot wait.

### Checks

- Separate stable decisions from design-dependent ones.
- Avoid pretending the handoff is fully complete.
- Keep the spec usable without hiding the unresolved design work.

## Cosmetic cleanup request edge path

### Prompt

Run Feature Spec Refiner when stakeholders ask for clearer wording but avoid addressing missing assumptions and edge behavior.

### Checks

- Refuse to reduce the work to cosmetic editing.
- Surface the missing assumptions directly.
- Reconnect clarity work to implementation risk.
