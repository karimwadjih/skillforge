# Examples

## Existing workflow under pressure

### Scenario

A team needs opportunity solution tree support for a real deadline instead of a brainstorming session.

### Prompt

Use Opportunity Solution Tree to produce an opportunity solution tree with goal alignment, opportunity branches, solution options, and assumption notes for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce an opportunity solution tree with goal alignment, opportunity branches, solution options, and assumption notes with explicit structure.
- Call out missing information and the next decision.
