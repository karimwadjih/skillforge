# Examples

## Existing workflow under pressure

### Scenario

A team needs source credibility audit support for a real deadline instead of a brainstorming session.

### Prompt

Use Source Credibility Audit to produce a credibility audit that tiers sources and explains how each should or should not influence a decision for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a credibility audit that tiers sources and explains how each should or should not influence a decision with explicit structure.
- Call out missing information and the next decision.
