# Source Credibility Audit

Audit a set of sources for credibility, bias, recency, and usefulness.

## Quick View

- Category: `research`
- Maturity: `experimental`
- Tier: `seeded`
- Evaluation status: `unbenchmarked`

## What This Skill Does

Teams frequently consume whatever sources are easiest to reach rather than what is most reliable.

## When To Reach For It

- The research set includes sources with mixed credibility.
- A team needs to know which inputs are strong enough to rely on.

## What You Should Provide

- source list
- decision context
- quality concerns

## What You Should Get Back

- credibility audit
- source tiers
- usage notes
