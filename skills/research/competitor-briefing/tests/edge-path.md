# Edge Path Tests

## Vaporware edge path

### Prompt

Run Competitor Briefing when the competitor announcement is polished but the product proof is thin, unclear, or impossible to verify.

### Checks

- Lower confidence instead of repeating the announcement at face value.
- Separate narrative power from product substance.
- Recommend follow-up evidence to watch for before changing plans.

## Segment mismatch edge path

### Prompt

Run Competitor Briefing when the move looks threatening but appears aimed at a buyer segment we rarely serve.

### Checks

- Test whether the segment overlap is real.
- Avoid inflating the internal threat level.
- Keep the response proportional to the actual overlap.
