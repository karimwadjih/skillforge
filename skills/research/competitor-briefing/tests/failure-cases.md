# Failure Case Tests

## Feature checklist failure probe

### Prompt

What does a poor Competitor Briefing look like when the output compares features carefully but says nothing about intent or response posture?

### Checks

- Identify the failure mode as comparison without strategy.
- Explain why the team still cannot decide what to do.
- State how the skill should reconnect competitor evidence to internal action.

## Mirror-response failure probe

### Prompt

What does a poor Competitor Briefing look like when the output assumes every visible competitor move requires an immediate match?

### Checks

- Identify the failure mode as strategic overreaction.
- Explain how that distorts roadmap or GTM decisions.
- State how the skill should reintroduce evidence thresholds and restraint.
