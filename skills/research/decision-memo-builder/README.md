# Decision Memo Builder

Shape research and context into a decision memo with options and recommendations.

## Quick View

- Category: `research`
- Maturity: `scaffold`
- Tier: `seeded`
- Evaluation status: `unbenchmarked`

## What This Skill Does

Teams gather enough information to decide but fail to package it into a format leaders can review quickly.

## When To Reach For It

- You need to turn research into a decision memo.
- Stakeholders need options, tradeoffs, and a recommendation in one place.

## What You Should Provide

- background
- decision statement
- evidence
- options

## What You Should Get Back

- decision memo
- option comparison
- recommendation
