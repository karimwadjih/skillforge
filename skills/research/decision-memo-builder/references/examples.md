# Examples

## Existing workflow under pressure

### Scenario

A team needs decision memo builder support for a real deadline instead of a brainstorming session.

### Prompt

Use Decision Memo Builder to produce a decision memo with background, options, tradeoffs, recommendation, and follow-up questions for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a decision memo with background, options, tradeoffs, recommendation, and follow-up questions with explicit structure.
- Call out missing information and the next decision.
