# Failure Case Tests

## Chronological reading dump failure probe

### Prompt

What does a poor Literature Map look like when the source list is long but structurally repetitive?

### Checks

- Identify the failure mode as a source list disguised as synthesis.
- Explain why the reader still lacks orientation.
- State how the skill should recover by clustering and prioritizing.

## False consensus failure probe

### Prompt

What does a poor Literature Map look like when the sources appear aligned only because the analysis erases the real disagreements?

### Checks

- Identify the failure mode as flattening live disagreements into fake agreement.
- Explain why that distorts downstream decisions.
- State how the skill should surface the contested questions directly.
