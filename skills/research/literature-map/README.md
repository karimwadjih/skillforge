# Literature Map

Map a crowded topic into schools of thought, evidence weight, and the shortest smart reading path.

## Quick View

- Category: `research`
- Maturity: `frontier`
- Tier: `developed`
- Evaluation status: `sampled`

## What This Skill Does

Teams often burn time reading source by source without first seeing the main schools of thought, the evidence hierarchy, and the disagreements that actually matter to the decision.

## When To Reach For It

- You need a fast orientation before a team spends a week reading the wrong sources.
- The literature is large, repetitive, or internally contested and you need the debate structure first.

## What You Should Provide

- framing question or decision the reading should support
- seed bibliography, search results, or must-read source set
- time window or domain boundary
- constraints on depth, rigor, or turnaround time

## What You Should Get Back

- literature landscape map
- theme clusters and fault lines
- evidence-strength notes
- recommended reading path

## Repository Notes

- Main instructions live in `SKILL.md`.
- Machine-readable metadata lives in `skill.json`.
- Examples, rubric, edge cases, and tests live under `references/` and `tests/`.
