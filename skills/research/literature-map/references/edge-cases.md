# Edge Cases

## Citation concentration masks a thin field

### Risk

The topic looks mature because many papers cite the same foundational work, but fresh evidence is scarce.

### How The Skill Should Respond

Call out citation concentration, separate foundational from recycled claims, and lower confidence where newer evidence is weak.

### Failure Probe

The response ignores the complication or hides the uncertainty instead of adapting the workflow.

## Different disciplines use incompatible vocabulary

### Risk

The same topic is described with different terms across fields, making the literature look more fragmented than it really is.

### How The Skill Should Respond

Normalize the terminology, note where the mapping is imperfect, and preserve the concept groups across naming differences.

### Failure Probe

The response ignores the complication or hides the uncertainty instead of adapting the workflow.

## Recency bias distorts importance

### Risk

Newer articles dominate attention even though older work still frames the core evidence better.

### How The Skill Should Respond

Separate recency from analytical weight and explain when an older source still matters more than the latest commentary.

### Failure Probe

The response ignores the complication or hides the uncertainty instead of adapting the workflow.
