# Examples

## AI evaluation methods scan

### Scenario

A research and product team needs to understand the main evaluation approaches behind agent benchmarks before committing to one direction.

### Prompt

Map the literature on rubric-based evaluation, model-graded evaluation, and human-review hybrids so the team can decide what to pilot first.

### Strong Response Should

- Show the major schools of thought rather than listing papers chronologically.
- Distinguish foundational work from recent but thin commentary.
- End with a reading path and the key disagreement the team still needs to resolve.

## Board diligence reading plan

### Scenario

Leadership needs a fast map of a new domain before a board discussion, but nobody has time to read twenty sources end to end.

### Prompt

Produce a literature map on supply-chain automation adoption so leadership can see the main themes, evidence quality, and what to read next.

### Strong Response Should

- Group the sources into recognizable themes with representative references.
- Call out where the evidence is mature versus where it is still speculative.
- Recommend the next three readings that would most change the decision posture.

## Cross-discipline terminology mess

### Scenario

An internal strategy group is reading healthcare operations, economics, and vendor material that describe the same coordination problem with different language.

### Prompt

Build a literature map on care-team workflow automation where the sources disagree on terminology, study design, and evidence quality.

### Strong Response Should

- Normalize the vocabulary without pretending the fields are identical.
- Keep evidence quality visible even when the source types vary sharply.
- Identify the shortest reading sequence for someone who needs a usable mental model fast.
