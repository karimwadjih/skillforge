# Examples

## Existing workflow under pressure

### Scenario

A team needs field note synthesizer support for a real deadline instead of a brainstorming session.

### Prompt

Use Field Note Synthesizer to produce a synthesis note with recurring themes, exceptions, evidence snippets, and suggested follow-up work for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a synthesis note with recurring themes, exceptions, evidence snippets, and suggested follow-up work with explicit structure.
- Call out missing information and the next decision.
