# Examples

## Existing workflow under pressure

### Scenario

A team needs safety case builder support for a real deadline instead of a brainstorming session.

### Prompt

Use Safety Case Builder to produce a safety case with hazards, controls, residual risks, and recommended monitoring for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a safety case with hazards, controls, residual risks, and recommended monitoring with explicit structure.
- Call out missing information and the next decision.
