# Safety Case Builder

Build a safety case for an agent workflow by mapping risks, controls, and residual uncertainty.

## Quick View

- Category: `frontier`
- Maturity: `experimental`
- Tier: `seeded`
- Evaluation status: `unbenchmarked`

## What This Skill Does

Teams often talk about safety controls informally rather than documenting the reasoning in one place.

## When To Reach For It

- A workflow needs a safety case.
- The team wants a more explicit map of controls and residual risks.

## What You Should Provide

- workflow
- risk concerns
- existing controls

## What You Should Get Back

- safety case
- risk map
- control summary
- residual risks
