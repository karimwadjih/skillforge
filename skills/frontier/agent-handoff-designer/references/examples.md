# Examples

## Existing workflow under pressure

### Scenario

A team needs agent handoff designer support for a real deadline instead of a brainstorming session.

### Prompt

Use Agent Handoff Designer to produce a handoff design with stage boundaries, context contract, failure modes, and fallback behavior for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a handoff design with stage boundaries, context contract, failure modes, and fallback behavior with explicit structure.
- Call out missing information and the next decision.
