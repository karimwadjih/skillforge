## Existing workflow under pressure

### Scenario

A team needs reasoning trace redactor support for a real deadline instead of a brainstorming session.

### Prompt

Use Reasoning Trace Redactor to produce a redaction plan with removable fields, share-safe summaries, and residual disclosure risks for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a redaction plan with removable fields, share-safe summaries, and residual disclosure risks with explicit structure.
