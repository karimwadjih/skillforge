# Prompt Regression Hunter

Hunt for regressions between prompt or skill versions using consistent comparisons.

## Quick View

- Category: `frontier`
- Maturity: `frontier`
- Tier: `developed`
- Evaluation status: `sampled`

## What This Skill Does

Prompt changes often improve the headline case while regressing in edge or formatting behavior.

## When To Reach For It

- A skill or prompt changed and regression checking is needed.
- Version comparisons need more than subjective side-by-side reading.

## What You Should Provide

- version A
- version B
- comparison cases
- scoring focus

## What You Should Get Back

- regression brief
- difference summary
- risk cases

## Repository Notes

- Main instructions live in `SKILL.md`.
- Machine-readable metadata lives in `skill.json`.
- Examples, rubric, edge cases, and tests live under `references/` and `tests/`.

## Quality Position

This Skill is meaningfully filled for v1, with stronger examples and tests than the broader seeded tier.
