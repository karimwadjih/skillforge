# Examples

## Existing workflow under pressure

### Scenario

A team needs prompt regression hunter support for a real deadline instead of a brainstorming session.

### Prompt

Use Prompt Regression Hunter to produce a regression brief with changed behavior, risk cases, and recommended follow-up evaluation for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a regression brief with changed behavior, risk cases, and recommended follow-up evaluation with explicit structure.
- Call out missing information and the next decision.

## Partial inputs

### Scenario

Inputs arrive incomplete, but the team still needs a useful first-pass artifact from prompt regression hunter.

### Prompt

Draft the strongest version of a regression brief with changed behavior, risk cases, and recommended follow-up evaluation using only the available inputs and list what is missing.

### Strong Response Should

- Separate knowns from assumptions.
- Prioritize the most decision-relevant gaps.
- Keep the output usable even when some inputs are missing.
