# Examples

## Existing workflow under pressure

### Scenario

A team needs multimodal task brief support for a real deadline instead of a brainstorming session.

### Prompt

Use Multimodal Task Brief to produce a multimodal task brief with modality roles, constraints, output structure, and failure notes for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a multimodal task brief with modality roles, constraints, output structure, and failure notes with explicit structure.
- Call out missing information and the next decision.
