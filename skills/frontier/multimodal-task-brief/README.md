# Multimodal Task Brief

Create a task brief for workflows that span text, images, or other media inputs.

## Quick View

- Category: `frontier`
- Maturity: `experimental`
- Tier: `seeded`
- Evaluation status: `unbenchmarked`

## What This Skill Does

Multimodal tasks often fail because the input contract is fuzzy and the task brief assumes too much.

## When To Reach For It

- A task involves text plus image or media inputs.
- The team needs a cleaner multimodal task brief.

## What You Should Provide

- task goal
- modalities involved
- constraints
- expected outputs

## What You Should Get Back

- task brief
- modality handling notes
- output contract

## Repository Notes

- Main instructions live in `SKILL.md`.
- Machine-readable metadata lives in `skill.json`.
- Examples, rubric, edge cases, and tests live under `references/` and `tests/`.

## Quality Position

This Skill is part of the seeded breadth layer. It is intentionally lighter than the developed and flagship tiers.
