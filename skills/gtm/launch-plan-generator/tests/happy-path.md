# Happy Path Tests

## Existing workflow under pressure happy path 1

### Prompt

Use Launch Plan Generator to produce a launch plan with sequencing, owners, deliverables, decision gates, and post-launch follow-up for the current workstream.

### Checks

- State the workflow objective in plain language.
- Produce a launch plan with sequencing, owners, deliverables, decision gates, and post-launch follow-up with explicit structure.
- Call out missing information and the next decision.

## Partial inputs happy path 2

### Prompt

Draft the strongest version of a launch plan with sequencing, owners, deliverables, decision gates, and post-launch follow-up using only the available inputs and list what is missing.

### Checks

- Separate knowns from assumptions.
- Prioritize the most decision-relevant gaps.
- Keep the output usable even when some inputs are missing.
