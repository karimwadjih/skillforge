# Failure Case Tests

## Generic pitch prep failure probe

### Prompt

What does a poor Sales Call Brief look like when it summarizes the account but could work for almost any customer?

### Checks

- Identify the failure mode as low-specificity sales wallpaper.
- Explain why the team still lacks call leverage.
- State how the skill should reconnect the brief to the actual account and stakeholder mix.

## One-audience failure probe

### Prompt

What does a poor Sales Call Brief look like when it treats executives, operators, and reviewers as one audience with one objection set?

### Checks

- Identify the failure mode as stakeholder flattening.
- Explain how that hurts conversation quality.
- State how the skill should separate stakeholder goals and likely resistance.
