# Examples

## Existing workflow under pressure

### Scenario

A team needs pipeline review prep support for a real deadline instead of a brainstorming session.

### Prompt

Use Pipeline Review Prep to produce a pipeline review brief with movement, blockers, confidence notes, and decisions needed for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a pipeline review brief with movement, blockers, confidence notes, and decisions needed with explicit structure.
- Call out missing information and the next decision.
