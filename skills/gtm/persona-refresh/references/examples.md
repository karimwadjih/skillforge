# Examples

## Existing workflow under pressure

### Scenario

A team needs persona refresh support for a real deadline instead of a brainstorming session.

### Prompt

Use Persona Refresh to produce a refreshed persona with role context, goals, blockers, buying triggers, and change notes for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a refreshed persona with role context, goals, blockers, buying triggers, and change notes with explicit structure.
- Call out missing information and the next decision.
