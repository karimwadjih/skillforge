# Persona Refresh

Refresh a persona artifact using newer research, pipeline signals, or product context.

## Quick View

- Category: `gtm`
- Maturity: `scaffold`
- Tier: `seeded`
- Evaluation status: `unbenchmarked`

## What This Skill Does

Personas tend to linger long after the underlying audience understanding has shifted.

## When To Reach For It

- Existing personas are out of date.
- Recent research or sales signals should update audience understanding.

## What You Should Provide

- current persona
- new evidence
- strategy context

## What You Should Get Back

- refreshed persona
- changes summary
- open questions
