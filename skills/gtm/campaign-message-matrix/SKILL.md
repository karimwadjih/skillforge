# Campaign Message Matrix

## Summary

Create a message matrix across audience segments, channels, and desired actions.

## When To Use

- A campaign needs channel-specific message planning.
- You need a message matrix with segment framing, channel adaptation, proof points, and calls to action instead of unstructured notes.
- The workflow repeats often enough to justify a reusable asset.

## Avoid When

- There is no defined audience, channel, or launch moment yet.
- You need final legal review or brand approval rather than a working draft.

## Inputs

- campaign objective
- audiences
- channels
- proof points

## Output Contract

- message matrix
- channel guidance
- CTA map

## Workflow

1. Confirm the decision context, audience, and missing inputs.
2. Build a message matrix with segment framing, channel adaptation, proof points, and calls to action using the available evidence and constraints.
3. Separate facts from assumptions and make confidence explicit.
4. Finish with next actions, open questions, or escalation notes.

## Deliverable Shape

A message matrix with segment framing, channel adaptation, proof points, and calls to action

## Guardrails

- Do not invent facts that are not present in the supplied materials.
- Make uncertainty explicit when inputs are incomplete or contradictory.
- Keep the output structured enough that another operator can reuse it.
