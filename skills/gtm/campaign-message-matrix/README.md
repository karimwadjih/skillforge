# Campaign Message Matrix

Create a message matrix across audience segments, channels, and desired actions.

## Quick View

- Category: `gtm`
- Maturity: `experimental`
- Tier: `seeded`
- Evaluation status: `unbenchmarked`

## What This Skill Does

Campaign messaging often drifts across channels because the core and adapted messages are not aligned explicitly.

## When To Reach For It

- A campaign needs channel-specific message planning.
- Teams need to align on core and adapted messages across audiences.

## What You Should Provide

- campaign objective
- audiences
- channels
- proof points

## What You Should Get Back

- message matrix
- channel guidance
- CTA map
