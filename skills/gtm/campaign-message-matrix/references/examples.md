# Examples

## Existing workflow under pressure

### Scenario

A team needs campaign message matrix support for a real deadline instead of a brainstorming session.

### Prompt

Use Campaign Message Matrix to produce a message matrix with segment framing, channel adaptation, proof points, and calls to action for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a message matrix with segment framing, channel adaptation, proof points, and calls to action with explicit structure.
- Call out missing information and the next decision.
