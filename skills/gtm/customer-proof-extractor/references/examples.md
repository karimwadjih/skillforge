# Examples

## Existing workflow under pressure

### Scenario

A team needs customer proof extractor support for a real deadline instead of a brainstorming session.

### Prompt

Use Customer Proof Extractor to produce a proof library with evidence statements, quote candidates, and relevance by audience for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce a proof library with evidence statements, quote candidates, and relevance by audience with explicit structure.
- Call out missing information and the next decision.
