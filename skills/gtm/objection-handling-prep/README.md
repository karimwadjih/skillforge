# Objection Handling Prep

Prepare objection handling guidance for sales, partnerships, or launch support teams.

## Quick View

- Category: `gtm`
- Maturity: `experimental`
- Tier: `seeded`
- Evaluation status: `unbenchmarked`

## What This Skill Does

Objection handling often becomes improvisation instead of a reusable operating asset.

## When To Reach For It

- A go-to-market team wants a better objection handling asset.
- Repeated buyer or partner objections need structured prep.

## What You Should Provide

- objection list
- product context
- proof points
- known constraints

## What You Should Get Back

- objection guide
- response options
- evidence notes

## Repository Notes
