## Existing workflow under pressure

### Scenario

A team needs objection handling prep support for a real deadline instead of a brainstorming session.

### Prompt

Use Objection Handling Prep to produce an objection handling guide with concern framing, response options, and proof guidance for the current workstream.

### Strong Response Should

- State the workflow objective in plain language.
- Produce an objection handling guide with concern framing, response options, and proof guidance with explicit structure.
