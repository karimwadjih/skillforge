# Flagship Criteria

Flagship Skills represent the strongest public examples in the Skillforge library. They are not automatically certified, but they should be materially stronger than the broader seeded library.

## A flagship Skill should have

- a clearly differentiated workflow
- a polished `SKILL.md`
- realistic examples, not toy prompts
- a structured rubric with evaluation dimensions
- edge cases and failure probes
- benchmark-ready prompts and scorecards
- strong README framing that explains when to use the Skill

## A flagship Skill should not have

- generic copy that could fit multiple categories
- filler examples
- vague success criteria
- unsupported claims about external compatibility or adoption
