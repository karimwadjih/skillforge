# Roadmap

## Phase 1: Scaffold

- Set up the monorepo, workspace tooling, root docs, certification standards, and seed-data structure
- Define the shared schemas and generation pipeline
- Establish CI, validation, and catalog generation

## Phase 2: Flagship Skills

- Fully develop the 12 flagship Skills
- Add stronger examples, rubrics, failure probes, and benchmark prompts
- Prepare a small subset for human review toward `certified`

## Phase 3: Extension and Studio

- Expand the meaningfully filled non-flagship library
- Build the docs site and studio MVP
- Improve browsing, comparison, packaging, and starter generation flows

## Phase 4: Community Contributions

- Harden contributor documentation and templates
- Add review paths for externally contributed Skills and collections
- Improve generated indexes, issue templates, and feedback loops

## Phase 5: Benchmark System

- Extend benchmark coverage beyond the flagship tier
- Add richer scorecards and adapter integrations
- Publish clearer certification evidence and review history
