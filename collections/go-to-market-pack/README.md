# Go-To-Market Pack

A GTM execution pack for launches, positioning, proof, and higher-quality revenue conversations.

## Audience

GTM leads, PMMs, and revenue teams

## Included Skills

- `launch-plan-generator`: Build a practical launch plan with sequencing, owners, messaging dependencies, and readiness checks.
- `positioning-statement-builder`: Build a positioning statement with target audience, problem, value, and differentiation.
- `sales-call-brief`: Prepare an account-specific call brief with stakeholder intent, likely objections, and the best next move.
- `campaign-message-matrix`: Create a message matrix across audience segments, channels, and desired actions.
- `customer-proof-extractor`: Extract credible customer proof points from interviews, calls, or success stories.
- `pipeline-review-prep`: Prepare a clean pipeline review with deal movement, risks, and leadership discussion points.

## Intended Use Cases

- Prepare better launches
- Strengthen positioning and account prep
- Improve revenue and pipeline operating reviews

## Quick Start

1. Lock positioning before building downstream message assets.
2. Use launch planning and proof extraction together for stronger GTM execution.
