# Ops Pack

Recurring operating-system skills for cadence design, leadership prep, escalation clarity, and weekly signal review.

## Audience

Operators, chiefs of staff, and operations leads

## Included Skills

- `weekly-ops-update`: Build a weekly operating update that highlights signal, risk, and follow-ups without reading like status spam.
- `executive-meeting-prep`: Prepare a leadership meeting brief with objectives, decisions, risks, and likely questions.
- `operating-rhythm-designer`: Design an operating rhythm with clear decision cadence, artifact discipline, and escalation paths.
- `kpi-anomaly-brief`: Explain a KPI anomaly with possible drivers, confidence, and recommended next checks.
- `vendor-evaluation-memo`: Evaluate a vendor option using criteria, risks, and adoption fit.
- `escalation-path-auditor`: Audit an escalation path for clarity, timing, and owner confusion.

## Intended Use Cases

- Tighten recurring reviews and updates
- Improve leadership decision prep
- Clarify escalation and operating ownership

## Quick Start

1. Start with Weekly Ops Update for recurring cadence.
2. Use Executive Meeting Prep and KPI Anomaly Brief for decision-heavy weeks.
