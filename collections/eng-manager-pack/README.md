# Engineering Manager Pack

A set for onboarding, architecture explanation, migration planning, and bug triage.

## Audience

Engineering managers and tech leads

## Included Skills

- `repo-onboarding`: Turn an unfamiliar codebase into a fast-start onboarding brief with setup, architecture, and first-task guidance.
- `architecture-explainer`: Explain a system or codebase architecture in a way that is grounded, digestible, and useful for future work.
- `bug-triage`: Turn a raw bug report into a triage artifact with severity, repro confidence, likely owners, and next actions.
- `migration-planner`: Plan a code, data, or platform migration with dependencies, sequence, and rollback considerations.
- `api-change-reviewer`: Review an API change for silent behavior shifts, consumer blast radius, and rollout realism.
- `test-plan-designer`: Design a test plan for a change, workflow, or system area.

## Intended Use Cases

- Accelerate team ramp-up
- Review changes and migrations
- Improve technical communication

## Quick Start

1. Use Repo Onboarding or Architecture Explainer for context building first.
2. Use Bug Triage or Migration Planner when a concrete engineering problem appears.
